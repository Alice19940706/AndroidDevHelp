# AndroidDevHelp
该项目用来解决android编程中疑难杂症的辅助仓库。灾难性的bug问题请发Issues！

在得到帮助的同时，希望大家能给予其他人帮助。相信在帮助他人的过程中也能同时收获成长。

# 如何求助
- 点击Issues 创建它。然后将你的问题描述仔细，若有错误代码务必贴上来。
